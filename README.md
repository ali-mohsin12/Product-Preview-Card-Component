## 🎯 What I Learned

In this project, I practiced building a responsive two-column layout and switching images for different screen sizes. I improved my understanding of Flexbox, spacing, and aligning content to closely match a design reference.

I also learned how to use media queries effectively and how to style interactive elements like buttons with hover states.

---

## 🚧 Challenges I Faced

One of the main challenges was handling the responsive image swap between desktop and mobile layouts. I overcame this by using separate image elements and controlling their visibility with media queries.

Another challenge was getting the spacing and alignment right inside the card. I adjusted padding, flex properties, and font sizes until the layout matched the design more closely.

## ⭐ What I’m Proud Of

I’m proud of the final layout and how closely it matches the design, especially the responsive behavior and clean visual structure.

---

## 🔄 What I Would Do Differently

Next time, I would focus more on writing reusable CSS and improving accessibility, such as better alt text and semantic HTML.
